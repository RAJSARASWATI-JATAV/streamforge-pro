
### Configuration File

Create `~/.streamforge/config/app_config.yml`:

```yaml

  output_directory: ~/Downloads/StreamForge
  max_concurrent: 5

  default_profile: balanced
  auto_select: true

  timeout: 30
  retries: 3
```
